# icepuma/taps

Homebrew tap for formulas maintained in this repository.

## Quick Start

```sh
brew tap icepuma/taps
brew install <formula>
```

## Formulas

| Formula | Repository | macOS | Linux |
| --- | --- | --- | --- |
| `nanite` | [`icepuma/nanite`](https://github.com/icepuma/nanite) | `arm64` | `arm64`, `x86_64` |

See [`Formula/nanite.rb`](Formula/nanite.rb) for formula details.
